# Yate
Yate stands for "Yet another text editor"
It's an extremely simple text editor for the terminal. 
WIP

# Building
Dependencies: 
* A C++17 compatible compiler
* The ncurses library
* [Cmake](https://cmake.org/)
```bash
mkdir build
cd build
cmake ..
make
./yate <file>
```
